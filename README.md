# 📚 ShelfWise

A full-stack **Book Catalog App** built with:

- **PostgreSQL** – Database  
- **Node.js + Express** – Backend API  
- **React (Vite)** – Frontend  
- **Tailwind CSS + DaisyUI** – Styling  

## 🚀 Features
- Add, update, delete, and browse books
- Responsive UI
- REST API integration
